# Design System Theme
 
This document outlines the core aesthetic and functional parameters of our design system.
 
## Color Palette
 
Our system employs a `light` color mode. The primary brand color is `#2563EB`, providing a strong and consistent identity. A supporting secondary color is `#475569`, used for less prominent elements. The base neutral color for backgrounds and surfaces is `#F8FAFC`.
 
## Typography
 
Our typographic system is built on a clear hierarchy:
*   **Headlines**: `manrope`
*   **Body Text**: `inter`
*   **Labels**: `inter`
 
## Shape and Form
 
User interface elements feature a `moderate` roundedness, providing a balanced and approachable aesthetic.
 
## Spacing
 
The design system utilizes `compact` spacing between and within UI elements, creating a focused and efficient layout.
